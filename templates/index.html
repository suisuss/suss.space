Extra

This isn't some pathetic Weebly drag and drop built site made over the course of a weekend. This website was built from the ground up.

SUSS.SPACE was developed with a lightweight WSGI web application framework known as 'Flask'. The Flask application is run with Gunicorn in conjunction with Nginx on a Linux server over in Singapore. I access the server via SSH connection with a dedicated Raspberry Pi 2 to modify the site and it's content.

I came up with the idea of setting a website up like this back in October of 2018, as an aspiring programmer I knew I could learn a lot from deploying a website like this. I started putting my thought into action when I finally got a clue as to what the site would represent and the content it would hold, in March of 2019.
